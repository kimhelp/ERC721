# OpenSea NFT

ERC20 > 라이브러리
라이브러리 먼저 만들어보고
NFT 구현

```
mkdir truffle

cd truffle

npx truffle init

```

# remix 설치

```
npm i -g @remix-project/remixd
cd contracts

npx remixd -s . --remix-ide https://remix.ethereum.org
```

### 터미널 하나 추가

```
cd truffle
npm init
npm i openzeppelin-solidity
```
